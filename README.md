# 🚀 AI Career Navigator
**Empowering Students with Personalized, AI-Driven Career Roadmaps.**

### 🌟 Project Overview
AI Career Navigator is a smart tool designed to solve the "Directional Paralysis" faced by students post-college and also 10th and 12th students. It generates a structured **30-day actionable roadmap** based on a user's interest and skill level, bridging the gap between learning and employment.

### 🏆 Hackathon Tracks
- **Theme:** EdTech
- **Problem Statement:** Economic Empowerment & Employment

### ✨ Key Features
- **Personalized Learning Paths:** Dynamically generated roadmaps using Claude 3 AI.
- **Resilient Architecture:** Features a fail-safe "Offline Knowledge Engine" to provide guidance even during API/Internet downtime.
- **Structured Milestones:** Breaks down goals into 4 clear weekly phases.
- **Ethical AI Design:** Built-in guardrails to ensure safe and professional career advice.

### 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Modern UI/UX), JavaScript (ES6+)
- **AI Model:** Anthropic Claude 3 Haiku (via API)
- **Deployment:** GitHub Pages

### 🛡️ Ethical Alignment
We prioritize user safety by:
1. Providing a clear **Disclaimer** that AI advice is for guidance and should be verified by professionals.
2. Instructing the AI to avoid "Get-rich-quick" schemes, focusing on skill-based growth.
3. Ensuring zero data collection for user privacy.

### 🚀 How to Run Locally
1. Clone the repository.
2. Open `index.html` in any modern browser.
3. *Note:* To bypass CORS restrictions on a local machine, use the "Allow CORS" browser extension.
